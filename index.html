Доступные фигуры:

  • line
  • rectangle
  • triangle
  • circle
  • ellipse

Доступные опции:

  • Для отрисовки нескольких фигур за раз используйте разделитель "&"
  • "-p" - координаты точек в формате [x1, y1] [x2, y2]
  • "-c" - цвет линии (rgb, rgba, hsl)
  • "-b" - цвет фона (rgb, rgba, hsl)
  • "-w" - толщина линии
  • "-r" - радиус (только для круга)
  • "-r1" и "-r2" - радиусы (только для эллипса)

Например:

  • rectangle -p [50, 50] [100, 100] -c lightblue -b rgba(255, 0, 0, 0.3) -w 5 & triangle -p [50, 50] [50, 100] [100, 100] -c #42d2eb -b rgba(232, 235, 66, 0.3)
  • circle -p [75, 75] -r 25 -c rgb(255, 0, 0) -b rgba(0, 255, 0, 0.3)
